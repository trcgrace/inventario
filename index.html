☁️ Conectar con Firebase (Gmail)

Pega los datos de tu proyecto Firebase. Se guardan en este dispositivo y no se comparten con nadie.

Cómo obtener estos datos (8 min · gratis con Gmail):
1Ve a firebase.google.com → clic en Ir a la consola → entra con tu Gmail
2Clic en Agregar proyecto → nombre: "Inventario" → desactiva Google Analytics → Crear
3En el panel del proyecto: Firestore Database → Crear base de datos → Modo prueba → Siguiente → Habilitar
4Clic en el ícono </> (app web) → registra la app → copia apiKey y projectId
5Pega ambos datos arriba y toca Conectar
💡 Los mismos datos en todos tus dispositivos = sincronización automática en tiempo real.
